# NTS_Intern
네이버 예약시스템

예약서비스란 ? 
- 네이버웹 서비스에서 사용됐던 서비스입니다.
- 크게 5가지 기능이 있습니다

1. 메인페이지
- 프로모션 : 현재 상영중인 연극들의 대표 이미지들을 슬라이드로 보여줍니다.
- 카테고리 : 상영중인 연극들을 분류하고, 원하는 종류의 연극을 따로 볼 수 있습니다.
- 상품리스트 : 연극들의 광고판과, 설명을 보여줍니다

2. 각 항목의 상세페이지
- 예매하기 : 해당 연극을 예매하는 페이지로 전환시켜 줍니다.
- 한줄평 내역 : 연극을 본 고객들의 후기 및 별점을 볼 수 있습니다.

3. 예약하기
- 티켓 수 선택 : 성인, 유아, 청소년, 특가를 선택하는 항목입니다.
- 예매자 정보 : 예매하는 예매자의 정보를 적는 항목입니다.
- 약관 동의 : 개인정보 수집 및 이용동의에 대한 설명 및 체크 문항입니다.
- 예약하기 버튼 : 사용자의 입력 내용을 기반으로, 예약을 하는 기능입니다.

4. 나의 예매 내역 확인
- 예약확인 : 이용예정, 이용완료, 취소&환불 에 대한 내역을 볼 수 있습니다.
- 상세 리스트 : 예약확인에서 버튼을 누르면, 각 해당 연극에 대한 일시, 장소, 내역, 금액을 볼 수 있습니다.
- 리뷰남기기 : 이용완료된 연극에 한에서, 리뷰를 남길 수 있는 페이지로 이동합니다.

5. 한줄평등록
- 별점 : 연극에 대한 만족도를 입력합니다.
- 리뷰 입력 : 연극을 보고 난 후의 소감을 적는 칸입니다.
- 리뷰 등록 : 입력한 리뷰를 등록하고, 나의 예매내역 확인 페이지로 이동합니다.
